# Nawaz Hotel Website

A simple, responsive hotel website built with HTML5, CSS3, and JavaScript, created for the MANTRA 2026 Summer School frontend assignment.

## Pages
- index.html - Home page with hero banner, hotel introduction, and highlights
- about.html - About the hotel, mission, values, and FAQ toggle
- rooms.html - Room categories, facilities, and pricing
- gallery.html - Filterable photo gallery
- booking.html - Room booking form with room cost calculator
- contact.html - Contact information and enquiry form

## Features
- Responsive layout using CSS media queries (mobile, tablet, desktop)
- Basic SEO: title tags, meta description, meta keywords, one h1 per page, alt text on images, canonical tags, robots.txt, and sitemap.xml
- JavaScript features: room cost calculator, booking form validation, gallery filter, and FAQ toggle
- Clean and modern user interface with consistent navigation and footer across all pages
- Favicon included
